#ວິທີໃຊ້ງານ
1 ໃຫ້ນັກສຶກສາ Fork ເອົາ Repository ນີ້
2 ໃຫ້ນັກສຶກສາຄົ້ນຄວ້າ ສຶກສາວ່າໄດ້ນຳໃຊ້ Package ໃດແດ່ທັງໃນ Front-end ແລະ Back-end
3 ເປີດ Termini ແລ້ວເຂົ້າໄປທີ່ Folder back-end - ຈາກນັ້ນສັ່ງ pnpm install - ຕໍ່ດ້ວຍຄຳສັ່ງ pnpm prisma migrate dev --name init ຈາກນັ້ນລອງກວດເບິ່ງຖານຂໍ້ມູນວ່າມີບໍ່ - ຕໍ່ດ້ວຍຄຳສັ່ງ pnpm prisma generate - ຈາກນັ້ນສັ່ງ pnpm dev ແລ້ວສັງເກດເບິ່ງວ່າມັນ run ທີ່ port ໃດ
4 ເປີດ Terminal ໃໝ່ ແລ້ວເຂົ້າໄປທີ່ Folder front-end - ຈາກນັ້ນສັ່ງ pnpm install - ເປີດຟາຍ .evn ຂອງ front-end ແລ້ວແກ້ port ໃຫ້ກົງກັບຂອງ Back-end ທີ່ກຳລັງ run ຢູ່ - ຈາກນັ້ນສັ່ງ pnpm dev ແລ້ວສັງເກີດເບິ່ງມັນຂຶ້ນ port ໃດ
5 ເປີດ Web Brower ແລ້ວເຂົ້າໜ້າ localhost:<port_front_end>/register - ຕົວຢ່າງ localhost:3002/register - ຈະເຫັນໜ້າ register ແລ້ວໃຫ້ທົດລອງລົງທະບຽນ - ຈາກນັ້ນແລ້ວກໍລອງ login ໂດຍເຂົ້າໄປທີ່ localhost:<port_fornt_end>/login
